# Algorithm Package Template

Use this directory as the starting point for a new plugin package.

Expected package structure:

- `pyproject.toml.example`: package metadata and plugin entry point declaration
- `src/algo_template/entry.py`: algorithm entry class
- `src/algo_template/meta.py`: algorithm metadata
- `src/algo_template/validators.py`: input and parameter validation
- `tests/test_entry.py`: basic template assertions
